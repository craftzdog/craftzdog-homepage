自分がプレイしているゲームや好きなゲームをTwitterのフォロワーに紹介するために制作したWebアプリケーション。カード形式でゲームを視覚的に表示し、フォロワーが一目で興味のあるゲームを見つけられるギャラリーサービス。

## 使用技術

- Next.js
- TypeScript
- Tailwind CSS

## 主な機能

- ゲームをカード形式で表示
- プラットフォーム別タグ表示（Switch、PC等）
- ユーザープロフィール機能
- レスポンシブデザイン（現在はスマートフォンのみ）
