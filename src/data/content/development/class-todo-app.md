Web初学者のためのバックエンド学習を目的として開発したTodoアプリケーション。Azure Functionsを使用したサーバーレスAPIとAzure SQL Databaseを組み合わせ、フロントエンドからバックエンドまでの一連の開発フローを実践。REST APIの設計・実装とデータベース連携の基礎を学ぶための教材として制作。

## 使用技術

**フロントエンド**

- Next.js
- JavaScript

**バックエンド**

- Azure Functions Core Tools

**クラウドサービス**

- Azure App Service
- Azure Functions
- Azure SQL Database

## 主な機能

- タスクの一覧表示
- タスクの追加・削除
- タスクの完了/未完了切り替え
- 残りタスク数の表示
- REST APIによるCRUD操作
- クラウドデータベースとの連携
